魔兽空格键默认功能是控制角色跳跃，但这太奢侈了。通过按键映射工具，可以让空格键既能跳跃，又能施法。

### 实现步骤

1. 将空格键绑定到某个动作条的快捷键上
2. 将高频使用的瞬发技能拖到此快捷键上，例如冰法的冰枪术
3. 将 100% 不会使用的组合键用来控制角色跳跃，例如「Alt+Ctrl+Shift+鼠标中键」
4. 用按键映射工具实现按一下空格，自动附加「Alt+Ctrl+Shift+鼠标中键」的效果

### 添加 Complex Modifications 规则

仍然使用前文提到的 karabiner-elements 实现本次目标，相应规则如下

```json
{
  "description": "Moonlight: Spacebar -> Spacebar + (Alt+Ctrl+Shift+Button3)",
  "manipulators": [
    {
      "conditions": [
        {
          "bundle_identifiers": ["^com\\.moonlight-stream\\.Moonlight$"],
          "type": "frontmost_application_if"
        }
      ],
      "from": { "key_code": "spacebar" },
      "to": [
        { "key_code": "spacebar" },
        {
          "modifiers": ["left_alt", "left_control", "left_shift"],
          "pointing_button": "button3"
        }
      ],
      "type": "basic"
    }
  ]
}
```

返回 Moonlight 进入魔兽世界，用冰法测试：

- 选中敌对目标时，按下空格可以在跳跃的同时施放冰枪术
- 选中无效目标或无目标时，按下空格可以跳跃

让空格键绑定一个纯技能仍然太奢侈，应该绑定一个宏命令。毕竟空格键又大又顺手，在保留跳跃功能的同时，肯定是要用宏多玩点花样的。

### 以冰法为例

空格键绑定的**跳跃宏**举例如下

```sh
#showtooltip
/stopmacro [mounted]
/cast [@mouseover,help,nodead] 解除诅咒; [mod] 寒冰护体; [@mouseover,harm,nodead][harm,nodead] 冰枪术; 奥术智慧
```

如果处于**乘骑状态**，则啥也不干。这是为了防止在骑马、大鸟或御龙飞行时，因施放技能导致坠机或下马。此时，按下空格键仅仅控制乘骑跳跃或提升大鸟高度。

如果不在乘骑状态，则按如下优先级施放技能：

1. 鼠标悬停在队友的**血条**或**物理模型**上（且队友没死），按下宏就会直接解诅咒。这个判定高居榜首，不需要你切目标，非常适合瞬间救场。
2. 如果鼠标没有指着友方，此时只要你按住了键盘上的任何修饰键（Shift、Ctrl 或 Alt）再按这个宏，就会直接给自己套冰盾。
3. 如果前两个条件都不满足，宏会开始判定敌对。这里做了双重智能判定：
   - 优先打**鼠标指向的敌对目标**（例如在无需切换当前目标的情况下打图腾，或者拉怪）。
   - 如果鼠标没指怪，就直接打**当前锁定的敌对目标**。

4. 如果以上三个战斗条件全都不满足（比如你空选目标、鼠标指着空地、或者队伍里没人在战斗），宏就会自动施放法师的常用常驻 Buff **奥术智慧**。

> [!TIP]
> 空格键绑定的必须是瞬发技能，因为读条技能会因为自己的跳跃行为而自行中断
