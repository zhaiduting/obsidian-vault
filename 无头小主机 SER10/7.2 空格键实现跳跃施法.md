魔兽空格键默认功能是控制角色跳跃，但这太奢侈了。通过按键映射工具，可以让空格键既能跳跃，又能施法。

### 实现步骤

1. 将空格键绑定到某个动作条的快捷键上
2. 将高频使用的瞬发技能拖到此快捷键上，例如冰法的冰枪术
3. 将 100% 不会使用的组合键用来控制角色跳跃，例如「Alt+Ctrl+Shift+鼠标中键」
4. 用按键映射工具实现按一下空格，自动附加「Alt+Ctrl+Shift+鼠标中键」的效果

### 添加 Complex Modifications 规则

仍然使用前文提到的 karabiner-elements 实现本次目标，相应规则如下

```json
{
  "description": "Moonlight: Spacebar -> Spacebar + (Alt+Ctrl+Shift+Button3)",
  "manipulators": [
    {
      "conditions": [
        {
          "bundle_identifiers": ["^com\\.moonlight-stream\\.Moonlight$"],
          "type": "frontmost_application_if"
        }
      ],
      "from": { "key_code": "spacebar" },
      "to": [
        { "key_code": "spacebar" },
        {
          "modifiers": ["left_alt", "left_control", "left_shift"],
          "pointing_button": "button3"
        }
      ],
      "type": "basic"
    }
  ]
}
```

返回 Moonlight 进入魔兽世界，用冰法测试：

- 选中敌对目标时，按下空格可以在跳跃的同时施放冰枪术
- 选中无效目标或无目标时，按下空格可以跳跃

让空格键绑定一个纯技能仍然太奢侈，应该绑定一个宏命令。毕竟空格键又大又顺手，在保留跳跃功能的同时，肯定是要用宏多玩点花样的。
