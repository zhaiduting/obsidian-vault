Window 应用程序只能使用 Alt / Ctrl / Shift 这 3 个物理修饰键，不能使用 Win 徽标键（系统占用），也不能识别苹果键盘的 Cmd 键。折中的办法是「将 Cmd 键映射成 Alt + Ctrl 的组合键」，这可以继续保持 Cmd 作为修饰键的功能。

### 安装 Mac 系统的按键映射工具

```sh
brew install --cask karabiner-elements
```

### 添加 Complex Modifications 规则

此规则的意思就是仅在 Moonlight 中将左 Cmd 键映射成 Ctrl + Alt 的组合

```json
{
  "description": "Moonlight: Left Command -> Ctrl + Alt",
  "manipulators": [
    {
      "conditions": [
        {
          "bundle_identifiers": ["^com\\.moonlight-stream\\.Moonlight$"],
          "type": "frontmost_application_if"
        }
      ],
      "from": { "key_code": "left_command" },
      "to": [
        {
          "key_code": "left_control",
          "modifiers": ["left_option"]
        }
      ],
      "type": "basic"
    }
  ]
}
```

### 与 LinearMouse 配合使用

前文已述 LinearMouse 可以解决「Shift + 滚轮」问题，此功能对于 Karabiner Elements 同样有用，因为这可以确保「Shift + Cmd + 滚轮事件」绑定新的技能！

如果关闭 LinearMouse 则只能绑定「Cmd + 滚轮事件」，而不能绑定「Shift + Cmd + 滚轮事件」
