在 Sunshine + Moonlight 投屏成功的状态下可以关闭 OBS 程序，但却不能拔掉读卡器！解决方法：在小主机里安装虚拟显示器的驱动程序。命令如下

```sh
winget install "Virtual Display Driver"
```

之所以要安装这玩意，是因为 Window 在没有显示器的情况下不会输出图形界面。使用 winget 而不使用 choco 的原因是 choco 并未收录这个包。

事实上执行完 `winget` 的命令后 VDD 仍然无法正常工作，此时手动打开文件夹 `C:\Users\zhaid\AppData\Local\Microsoft\WinGet\Packages` 里面虚拟显卡驱动相关的子文件夹，然后双击其中的 `VDD Control.exe` 就会显示一个窗口

![VDD](https://lib.zhaiduting.work.gd/uPic/VDD.jpg)

再点击 VDD 控制界面的「Install Driver」一次，之后就可以正常拔掉读卡器了。此时的小主机与笔记本之间使用的是无线连接，真正实现了无头运行状态。
![VDD|](https://lib.zhaiduting.work.gd/uPic/VDD.png)

### 自定义分辨率

安装后的 VDD 默认只会读取最基础的几组分辨率，通过修改
`C:\VirtualDisplayDriver\vdd_settings.xml`
可以实现分辨率的自定义。试用以下内容**完全覆盖**你原来的 `vdd_settings.xml` 文件：

```xml
<?xml version='1.0' encoding='utf-8'?>
<vdd_settings>
    <monitors>
        <count>1</count>
    </monitors>
    <gpu>
        <friendlyname>default</friendlyname>
    </gpu>
    <global>
        <!-- 全局刷新率：下面添加的所有分辨率都会自动支持这些刷新率（包括 120Hz ProMotion） -->
        <g_refresh_rate>60</g_refresh_rate>
        <g_refresh_rate>90</g_refresh_rate>
        <g_refresh_rate>120</g_refresh_rate>
        <g_refresh_rate>144</g_refresh_rate>
        <g_refresh_rate>165</g_refresh_rate>
        <g_refresh_rate>244</g_refresh_rate>
    </global>
    <resolutions>
        <!-- ========================================== -->
        <!-- 一、【全屏模式推荐】 (14寸 Mac 独有 15.4:10 比例) -->
        <!-- 提示：UI 界面中看到高度为 1964、1664、1169、982 的即为全屏 -->
        <!-- ========================================== -->
        <resolution>
            <width>3024</width>
            <height>1964</height>
        </resolution>
        <resolution>
            <width>2560</width>
            <height>1664</height>
        </resolution>
        <resolution>
            <width>1800</width>
            <height>1169</height>
        </resolution>
        <resolution>
            <width>1512</width>
            <height>982</height>
        </resolution>

        <!-- ========================================== -->
        <!-- 二、【窗口模式推荐 - 16:10 比例】 (贴合 Mac 窗口生态) -->
        <!-- 提示：UI 界面中看到高度为 1800、1600、1200、1050 的即为窗口 -->
        <!-- ========================================== -->
        <resolution>
            <width>2880</width>
            <height>1800</height>
        </resolution>
        <resolution>
            <width>2560</width>
            <height>1600</height>
        </resolution>
        <resolution>
            <width>1920</width>
            <height>1200</height>
        </resolution>
        <resolution>
            <width>1680</width>
            <height>1050</height>
        </resolution>
        <resolution>
            <width>1440</width>
            <height>900</height>
        </resolution>

        <!-- ========================================== -->
        <!-- 三、【窗口模式推荐 - 16:9 标配宽屏】 (常规备用) -->
        <!-- ========================================== -->
        <resolution>
            <width>3840</width>
            <height>2160</height>
        </resolution>
        <resolution>
            <width>2560</width>
            <height>1440</height>
        </resolution>
        <resolution>
            <width>1920</width>
            <height>1080</height>
        </resolution>
        <resolution>
            <width>1366</width>
            <height>768</height>
        </resolution>
        <resolution>
            <width>800</width>
            <height>600</height>
        </resolution>
    </resolutions>
    <options>
        <CustomEdid>false</CustomEdid>
        <PreventSpoof>false</PreventSpoof>
        <EdidCeaOverride>false</EdidCeaOverride>
        <HardwareCursor>true</HardwareCursor>
        <SDR10bit>false</SDR10bit>
        <HDRPlus>false</HDRPlus>
        <logging>false</logging>
        <debuglogging>false</debuglogging>
    </options>
</vdd_settings>
```

修改分辨率后需要重启一下无头小主机才能生效。
