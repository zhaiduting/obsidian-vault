在 Sunshine + Moonlight 投屏成功的状态下可以关闭 OBS 程序，但却不能拔掉读卡器！解决方法：在小主机里安装虚拟显示器的驱动程序。命令如下

```sh
winget install "Virtual Display Driver"
```

之所以要安装这玩意，是因为 Window 在没有显示器的情况下不会输出图形界面。使用 winget 而不使用 choco 的原因是 choco 并未收录这个包。

事实上执行完 `winget` 的命令后 VDD 仍然无法正常工作，此时手动打开文件夹 `C:\Users\zhaid\AppData\Local\Microsoft\WinGet\Packages` 里面虚拟显卡驱动相关的子文件夹，然后双击其中的 `VDD Control.exe` 就会显示一个窗口

![VDD](https://lib.zhaiduting.work.gd/uPic/VDD.jpg)

再点击 VDD 控制界面的「Install Driver」一次，之后就可以正常拔掉读卡器了。此时的小主机与笔记本之间使用的是无线连接，真正实现了无头运行状态。
