在局域网 Wi-Fi 环境下（尤其是高性能路由器），不同投屏方案的延迟表现大概如下：

| **方案**                 | **延迟等级**      | **帧率/画质**    | **适用场景**                           |
| ------------------------ | ----------------- | ---------------- | -------------------------------------- |
| **Sunshine + Moonlight** | **极低 (5-15ms)** | **4K 60/120fps** | **首选方案，几乎无感操作**             |
| **NDI (OBS)**            | 中低 (30-70ms)    | 取决于设置       | 专业视频传输，对电脑性能有一定要求     |
| **AirPlay (模拟器)**     | 中 (80-150ms)     | 1080p 30/60fps   | 办公演示、看视频，不适合操作性强的游戏 |
| **Spacedesk/AweSun**     | 高 (100ms+)       | 波动较大         | 临时远程协助或作为静态副屏             |

结论很直接：**Sunshine + Moonlight 是目前无线投屏中速度最快、延迟最低的方案。**

### 安装命令

Mac 端安装 Moonlight

```sh
brew install --cask moonlight
```

SER10 (Windows) 端安装 Sunshine

```sh
choco install sunshine
```

SER10 (Windows) 端还需要安装虚拟外设驱动。如果不安装它，在 Mac 端连接的手柄将无法被 SER10 识别。

```sh
choco install vigembus
```

### 关键配置步骤 (避坑指南)

安装完成后，需要进行简单的配对才能跑起来：

1. **启动 Sunshine：** 安装后在 SER10 上启动 Sunshine。它通常会在后台运行，你可以通过浏览器访问 `https://localhost:47990` 进入后台管理页面。
2. **设置用户：** 第一次进入网页会要求设置用户名和密码（用于以后远程管理）。
3. **配对：**
   - 在 Mac 上打开 **Moonlight**，你应该能看到局域网里出现的 SER10 图标。
   - 点击图标，Mac 屏幕上会出现一组 **PIN 码**。
   - 回到 SER10 的 Sunshine 网页后台，点击导航栏的 `PIN`，输入这组数字即可完成握手。

4. **游戏手柄/鼠标优化：** 玩《魔兽世界》时，建议在 Moonlight 的设置中开启 **"优化鼠标进行远程桌面游戏"**。
   - 由于你使用的是 M2 MacBook Pro，可以在 Moonlight 设置里勾选 **"使用高效视频编码 (HEVC)"**，这能显著降低带宽压力并保持画质。

### 主要快捷键

以 Mac 系统为例  
退出 Moonlight 会话：⌃ + ⌥ + ⇧ + q  
全屏窗口切换：⌃ + ⌥ + ⇧ + x  
释放鼠标：⌃ + ⌥ + ⇧ + z  
性能统计数据显示切换：⌃ + ⌥ + ⇧ + s
