所谓「盲操关机」，指的是：在不看显示器的情况下，通过点鼠标或者敲键盘进行安全关机的操作。

### 盲操关机的应用场景

其实就是用来应急的：

- 当 Moonlight 的连接出现 Bug 时，需要重启电脑才能解决。通过盲操关机，可以实现安全重启。
- 当 Mac 已经关机了，突然发现无头主机还在运行。通过盲操关机可以立即将其关闭，无需再次打开 Mac 进入 Moonlight 去关机。

### 操作步骤

以罗技 WAVE 键盘为例，首先切换蓝牙键盘通道，使之与无头主机相连，然后再尝试以下步骤。

> [!CAUTION]
> 不要在 Moonlight 中验证以下操作，必须将键盘与无头主机直接相连才有效。因为当 Bug 发生时 Moonlight 的连接已经断开了！
>
> 以下操作对于锁屏界面无效！

当多媒体键处于关闭状态时

1. 连续多按几次 ⌘ + F4；
2. 按一次 Esc 键；
3. 再按一次 ⌘ + F4；
4. 直接敲击回车键关机。

如果以上操作不能关机，估计是多媒体键处于打开状态了。可以改用以下步骤再次尝试关机：

1. 连续多按几次 ⌘ + F4 + Fn；
2. 按一次 Esc 键；
3. 再按一次 ⌘ + F4 + Fn；
4. 直接敲击回车键关机。

### 步骤说明

如果打开了多个窗口，每按一次 ⌘ + F4 都会关闭一个当前窗口，多按几次的目的是将这些窗口全部关闭。当所有窗口都已关闭，继续按下 ⌘ + F4 时就会打开关机对话框，再次按下则关闭对话框。

因为处于无显示器状态，无法得知关机对话框到底有没有打开。通过按一下 Esc 键，我们可以明确判断关机对话框已被关闭。

当进行步骤3时，必定会打开关机对话框，最后第4步必定关机。

### 关于多媒体键

键盘最上面一排的 F1 到 F12 全部都是多媒体键。如果单独按下 F10 就可以实现静音效果，即可判断多媒体键处于打开状态。如果需要同时按下 Fn + F10 才能实现静音，则处于关闭状态。记住：关机对话框与多媒体键的状态刚好是相反的。

罗技 WAVE 键盘可以通过 Fn + Esc 键打开或者关闭多媒体键。
