可以按照以下步骤操作：

1. **打开终端。**
2. **进入 Python 环境：** 输入 `python3` 并按回车。
3. 输入代码。

## 示例

```sh
~ > python3
Python 3.9.6 (default, Dec  2 2025, 07:27:58)
[Clang 17.0.0 (clang-1700.6.3.2)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> ord('a')
97
>>> ord('A')
65

>>> ord('好')
22909
>>> hex(ord('好'))
'0x597d'
>>> hex(22909)
'0x597d'

>>> chr(46)
'.'
>>> ord('.')
46
>>> exit()
~ >
```

> [!TIP]
> 进入 Python 环境；
> 使用 ord() 查询 Unicode 编码；
> 使用 hex() 转换为十六进制

## 函数名

### 1. `ord` —— **Ordinal** (序数)

- **含义**：在数学和计算机科学中，**Ordinal** 表示一个元素在序列中的位置。
- **逻辑**：`ord('.')` 返回的是该字符在 **Unicode/ASCII 编码表** 中的“序号”或“位置”。例如，点号 `.` 在表中的第 46 个位置，所以结果是 46。

### 2. `chr` —— **Character** (字符)

- **含义**：这个缩写非常直观，就是“字符”本身。
- **逻辑**：它的功能是把一个数字（编码）转换回它所代表的 **Character**（字符）。

### 3. hex —— **Hexadecimal**

 /ˌheksə'desɪml/

- **Hexa-** (来自希腊语)：表示 **6**。
- **Decimal** (来自拉丁语)：表示 **10**。
- 合在一起就是 **16**，即我们常说的 **“十六进制”**。
