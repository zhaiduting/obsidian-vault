这种结构体没有显式的字段名，只有类型。

- **引用方式：** 使用索引，即 `self.0`, `self.1`, `self.2` ...
- **适用场景：** 当你想给一个基础类型（如 `u32`）起一个有意义的名字（包装模式），但字段本身不需要名字时。

举个例子，假设我们有一个 `Power` 结构体，代表角色的战斗力，我们希望两个角色相加能得到总战斗力。

```rust
use std::ops::Add;

struct Power(u32);

// 为 Power 实现 Add 特征
impl Add for Power {
    type Output = Power; // 相加的结果还是 Power 类型

    fn add(self, other: Power) -> Power {
        Power(self.0 + other.0)
    }
}

fn main() {
    let p1 = Power(50);
    let p2 = Power(100);

    let total = p1 + p2; // 就像加数字一样简单！
    println!("Total power: {}", total.0);
}
```

输出结果为 `Total power: 150`
