用于表示「任何实现了特定 trait 的类型」的抽象类型。 它允许在运行时存储和操作不同类型的值，只要这些类型实现了相同的 trait。以下代码中的 animal 就是特征对象类型（准确的说是：特征对象的不可变引用类型）

```rust
trait Speaker {
    fn speak(&self);
}

struct Dog;
impl Speaker for Dog {
    fn speak(&self) { println!("Woof!"); }
}
struct Cat;
impl Speaker for Cat {
    fn speak(&self) { println!("Mew!"); }
}

fn main() {
    let mut animal: &dyn Speaker = &Dog;
    animal.speak();
    animal = &Cat;
    animal.speak();

    let dog = &Dog;
    dog.speak();
    // dog = &Cat; // ❌ 类型错误
}
```
