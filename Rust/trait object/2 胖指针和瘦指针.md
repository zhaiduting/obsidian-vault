上回笔记[特征对象](1%20特征对象.md)的主函数里有一行赋值语句

```rust
let mut animal: &dyn Speaker = &Dog;
```

`&Dog` 是指向 Dog 的指针，这是一个瘦指针（Thin Pointer），只占一个 `usize`
`animal` 是一个胖指针（Fat Pointer），由两个 `usize` 组成：

1. **`data_ptr`**（指向实例数据）：**8 字节**。
2. **`vtable_ptr`**（指向虚函数表）：**8 字节**。

内存占用测试

```rust
println!("usize 的大小:      {} 字节", size_of::<usize>());
println!("&Dog (普通引用):   {} 字节", size_of::<&Dog>());
println!("&dyn Speaker (胖): {} 字节", size_of::<&dyn Speaker>());

/* 64 位机输出如下
usize 的大小:      8 字节
&Dog (普通引用):   8 字节
&dyn Speaker (胖): 16 字节
*/
```
