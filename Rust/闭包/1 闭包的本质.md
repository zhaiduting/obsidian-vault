> [!TIP]
> 本质是一个捕获了变量且实现了函数特征的结构体的实例

对于以下代码

```rust
let s = String::from("hello");
let c = || println!("{s}");
```

### 编译期

编译器自动完成以下步骤：

#### 1️⃣ 生成 struct

```rust
struct Closure<'a> {
    s: &'a String,
}

// 'a 是生命周期参数
// 表示这个 Closure 里面的引用 &String 能活多久
```

#### 2️⃣ 自动实现函数特征

根据变量的使用方式，此结构体会实现 `Fn / FnMut / FnOnce` 其中之一。本例实现的是 Fn 特征：

```rust
impl<'a> Fn() for Closure<'a> {
    fn call(&self) {
        println!("{}", self.s);
    }
}
```

#### 3️⃣ 自动创建实例

```rust
let c = Closure {
    s: &s,
};
```

生命周期参数 `'a` 被**具体化**了。它被绑定到变量 `s` 的作用域上。编译器会检查 `c` 的作用域。如果 `c` 试图被移动到一个比 `s` 活得更长的地方（例如返回给函数调用者），编译器会发现：`c` 内部引用的 `s` 已经失效了，从而报错。

### 运行时

当你通过 `c()` 执行闭包，实际上执行的是 `(&c).call()`，也就是 `Fn::call(&c)`
