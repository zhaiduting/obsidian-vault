```rust
fn main() {
    let my_plan = || {
        println!("闭包直到调用时才会执行");
        100
    };
    // 此时屏幕上什么都不会打印！
    // 只有当你执行 my_plan() 时，它才会运行。

    let my_result = {
        println!("代码块已经执行！");
        200
    };
    // 此时屏幕上立即显示打印结果

    println!("代码块的返回值为 {my_result}");
    println!("闭包的返回值为 {}", my_plan());
}
```

执行结果为

```
代码块已经执行！
代码块的返回值为 200
闭包直到调用时才会执行
闭包的返回值为 100
```

> [!TIP]
> 代码块立即执行，而闭包只有在调用时才会执行
> 它们都可以带有返回值

对于只有一行代码的闭包可以省略花括号

```rust
// 这样是可以的，简洁且合法
let square = |x: i32| x * x;

// 即使无参数也可以
let say_hi = || println!("Hi");

// 自动推导类型的多参数闭包
let add = |a, b, c| a + b + c;
let result = add(1, 2, 3);
println!("结果是: {}", result); // 6
```
