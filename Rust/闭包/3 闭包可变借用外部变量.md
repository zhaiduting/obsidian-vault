```rust
fn main() {
    let mut list = vec![1, 2, 3];
    println!("Before defining closure: {list:?}");

    let mut borrows_mutably = || list.push(7); // 可变借用 list
    // list.push(8); // ❌ 不可再借给其它代码
    // println!("Before calling closure: {list:?}"); // ❌ 同理不可借
    borrows_mutably(); // ✅
    borrows_mutably(); // ✅ 归还 list
    println!("After calling closure: {list:?}"); // ✅ 可借
}
```

闭包 `|| list.push(7)` 定义时，以可变借用的方式捕获 list 变量。最后一次执行 `borrows_mutably()` 时才会归还 list 的所有权。

归还之后的 `println` 可以正常借用 list

在 list 被借出后、尚未归还前的这段时间内，其它任何代码都不能再次借用 list，即使是只读借用也不行。
