先安装 RustRover

```sh
brew install --cask rustrover
```

然后通过 RustRover 安装 Rust

![安装Rust](https://resources.jetbrains.com/help/img/idea/2025.3/rust_no_toolchain_projectwizard_dark.png)

如果今后想要升级 Rust 可以手动敲入以下命令

```sh
rustup update
rustc --version # 查看编译器版本
```

参考链接 https://www.jetbrains.com/help/rust/rust-toolchain.html#install-rust-from-ide
