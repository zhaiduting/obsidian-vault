本仓库多设备同步

- 在 MacBook 的 obsidian 中编写笔记；
- 在 MacBook 的终端敲入 `git push` 命令将笔记上传到 github.com 或者 gitee.com 的笔记仓库里；
- 手机端使用 obsidian 的 Git 插件拉取 github.com 的笔记仓库。

### 关键

电脑端的 `obsidian-vault/.git/config` 中使用的 url 如下

```
[remote "origin"]
	url = git@github.com:zhaiduting/obsidian-vault.git
	fetch = +refs/heads/*:refs/remotes/origin/*
```

手机端要改成如下 https 形式的网址（其它地方无需修改）

```
[remote "origin"]
	url = https://github.com/zhaiduting/obsidian-vault.git
	fetch = +refs/heads/*:refs/remotes/origin/*
```

### 手机端

手机端 obsidian 的 Git 插件仅仅设置两个地方

- Author name for commit: 随便填写
- Author email for commit: 这个似乎也可以随便写
- Password/Personal access token: 不用填的

这样就可以在手机里拉取仓库啦！

> [!TIP]
> 手机端只读不写，意味着只拉取仓库，不推送

### 实现手机端推送

此时才需要设置手机端 obsidian 的 Git 插件的 Personal access token（比较麻烦，因为需要事先在 Github 设置有效的私人访问令牌）
