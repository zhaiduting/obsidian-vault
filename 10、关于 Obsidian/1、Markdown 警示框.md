### 关于 Alert

这是一个非常实用的 Markdown 扩展语法，通常被称为 **Alerts（警示框）** 或 **Callouts（标注）**。并非 markdown 官方语法，而是 **GitHub Flavored Markdown (GFM)** 的扩展。GitHub 在 2023 年正式标准化了这种写法，以便在文档和 Issue 中提供更好的视觉提示。
