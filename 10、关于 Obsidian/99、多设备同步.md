假设在家时用电脑版的 obsidian 写笔记，外出后需要在手机版的 obsidian 中查看，这就涉及到笔记在多个设备之间进行同步的问题。方式有多种，本文简单介绍「Github 仓库+ Obsidian Git Plugin」实现同步的概要。

### 创建 Github 仓库

略

### 安装 Obsidian 及其插件

- 电脑和手机里都安装 Obsidian
- 电脑打开 Obsidian 后安装 Git 插件，但是手机里无需安装此插件

### 将电脑里的笔记上传到 Github 仓库

在 Obsidian 里面随便写篇测试用的笔记，然后将其上传到 Github 仓库。无论是用 Git Gui 还是用 Obsidian Git 插件，都可以。

### 将电脑里的笔记发送到手机里

推荐使用[多设备通信工具 LocalSend](https://localsend.org/download)完成这步，改用其它方式也行的。

这也会将之前在电脑端安装好的 Obsidian Git 插件一并发送到手机里。

### 配置手机端 Obsidian Git 插件

![配置obsidian的Git插件](https://lib.zhaiduting.work.gd/uPic/%E9%85%8D%E7%BD%AEobsidian.j%E7%9A%84Git%E6%8F%92%E4%BB%B6pg.jpg)

如图，手机端只需设置4个地方：

- Github 的用户名
- Github 的访问令牌（关键）
- 作者名（随意）
- 作者邮箱（随意）

### 手机端进行同步

在手机里尝试拉取 Github 仓库的文件，得到错误提示，大致是插件不支持 SSH 方式访问仓库。

解决方法：

- 手机文件管理器开启「显示隐藏文件」的功能
- 手机文件管理器打开 LocalSend 接收到的笔记仓库文件夹
- 修改此文件夹里的 .git/config 文件

首先找到如下所示的 "origin" 下方的 url

```
[remote "origin"]
	url = git@github.com:zhaiduting/obsidian-vault.git
```

修改成下面这样（只改链接）

```
[remote "origin"]
	url = https://github.com/zhaiduting/obsidian-vault.git
```

保存后再次打开 obsidian，此时即可通过 Git 插件完成同步。

### 测试

- 在电脑里修改测试用的笔记
- 将修改的结果上传到 Github 仓库
- 在手机里使用 Obsidian Git 插件拉取 Github 仓库的内容

如果能从手机里看到电脑里修改后的笔记就成功了。

### 特点

安卓手机无需借助 Termux 一样可以设置 Github 的仓库地址。
