顺便展示一下 headerTitle 和 title 的区别，以及两者都不存在时又会怎样（如 Messages 那样）

```jsx
const HomeTabs = createBottomTabNavigator({
	screens: {
		Feed: {
			screen: FeedScreen,

			options: {
				headerTitle: "headerTitle of Feed",
				title: "title of Feed",
			},
		},

		Messages: MessagesScreen,
	},
});
```
