在全局环境下定义一个元组结构体，用于保存 App 的标题状态

```rust
#[derive(Clone)]
struct TitleState(String);
```

在 App 组件内初始化标题

```rust
#[component]
fn App() -> Element {
    use_context_provider(|| TitleState("Hot Dog!".into()));
    rsx! {
        Title {}
    }
}
```

在 Title 组件内读取标题

```rust
#[component]
fn Title() -> Element {
    let TitleState(title) = use_context(); // ✅ 优雅写法
    rsx! {
        div {
            h1 { "{title}" }
        }
    }
}
```

也可以采用丑陋的`::<T> + t.0`写法，如下

```rust
    let title = use_context::<TitleState>(); // 👎
    rsx! {
        h1 { "{title.0}" }
    }
```
