终端敲入 `dx serve` 命令，默认情况下会显示一个链接，点击后可以在浏览器里查看效果。

### 优化开发体验 1

在 Mac 里，[使用台前调度分组 App](../Mac/使用台前调度分组%20App.md) 将 IDE 与浏览器拉进同一个组。

### 优化体验 2

如果不想用浏览器，可以改用桌面 App 进行预览。只需修改 `Cargo.toml` 文件的 `features.default` 值，如下：

```toml
[features]
default = ["desktop"]
```

此时敲入 `dx serve` 命令将会打开 `Dioxus App` 窗口（置顶显示）

### 优化体验3

如果想要禁止 `Dioxus App` 窗口的置顶功能，只需在终端敲入以下命令一次就可以永久生效

```sh
dx config set always-on-top false
```

再次敲入 `dx serve` 命令，打开的 `Dioxus App` 窗口就不会置顶了。对于非置顶状态的窗口可以[使用台前调度分组 App](../Mac/使用台前调度分组%20App.md) 将此窗口与 IDE 拉进同一个组

> [!TIP]
> ⌘ + 反引号键（\`）：在 IDE 与 `Dioxus App` 之间切换（组内切换）
> ⌘ + tab 键：从 IDE & `Dioxus App` 组切换到其它 App，再按则切回分组（组外切换）
