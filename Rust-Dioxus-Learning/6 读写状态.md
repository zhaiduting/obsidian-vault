统一使用 `use_signal(|| {...})` 初始化状态，参数皆为闭包。简单起见，以下组件都省略了外壳代码 `#[component] fn MyComponent() -> Element {...}`

### 字符串类型

```rust
let mut title = use_signal(|| "Dioxus 状态实验，点击可颠倒 🎯".to_string());
rsx! {
    h1 {
        onclick: move |_| {
            let mut t = title.write();
            *t = t.chars().rev().collect();
        },
        "{title}"
    }
}
```

### 数字类型

以下 `.with_mut()` 是老式写法，新版推荐 `.write()`

```rust
let mut count = use_signal(|| 0);
rsx! {
    p { "当前计数: {count}" }
    button { onclick: move |_| *count.write() += 2, "数字 +2" }
    button { onclick: move |_| count.with_mut(|c| *c += 1), "数字 +1" } //🚫 
}
```

### 结构体类型

使用 `.read()` 读取结构体

```rust
let mut user = use_signal(|| User {
    name: "Alice".into(),
    age: 25,
});
rsx! {
    h3 { "结构体 (Struct)" }
    p { "姓名: {user.read().name}" }
    p { "年龄: {user.read().age}" }
    button { onclick: move |_| user.write().age += 1, "年龄 +1" }
}
```

### 函数类型

方法的初始化涉及多层嵌套，最外层使用 `use_signal()` 包裹闭包；中间使用 `Box::new()` 包裹闭包；最内层才是实际想要初始化的闭包函数。

```rust
// 函数：必须包装在 Box<dyn Fn()> 中，解决闭包匿名类型不匹配问题
let mut action = use_signal::<Box<dyn Fn()>>(|| {
    Box::new(|| info!("初始动作：日志已打印"))
});

rsx! {
    h3 { style: "color:red", "闭包函数 (Boxed Closure)" }
    button { onclick: move |_| action.read()(), "输出信息到控制台" }
    button {
        // 写：替换函数时也需要 Box::new
        onclick: move |_| action.set(Box::new(|| info!("新动作：新日志！"))),
        "切换为新函数"
    }
}
```
