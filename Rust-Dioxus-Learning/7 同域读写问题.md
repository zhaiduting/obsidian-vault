以下代码在 state 处于读保护期间尝试修改它，代码报错 `error[E0502]`，无法编译。

```rust
/* // ❌
let cur = state.read();
*state.write() = *cur + 1;
// */
```

### 解决方式 1

使用 clone() 解决读、写保护冲突问题

```rust
let cur = state.read().clone();
*state.write() = cur + 1;
```

### 解决方式 2

使用代码块 `{...}` ⭐️

```rust
let cur = { *state.read() };
*state.write() = cur + 1;
```

### 解决方式 3

使用 drop() 立即释放读保护

```rust
let cur = state.read();
let c = *cur;
drop(cur);
*state.write() = c + 1;
```
